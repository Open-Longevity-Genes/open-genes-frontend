API проекта возвращает данные в формате json. Все эндпоинты принимают параметр ?lang (en|ru).

/api/ — получение общего списка всех занесенных в базу генов с краткой информацией по каждому гену:

/api/gene/<id> — получение полной информации об одном гене по его id.

/api/by-functional-cluster/<id>,<id>,... — получение списка генов, принадлежащих одновременно ко всем перечисленным функциональным кластерам. Структура ответа такая же, как в /api/.

/api/by-expression-change/<expression-change> — получение списка генов, имеющих заданное значение изменения экспрессии Структура ответа такая же, как в /api/.
