API проекта возвращает данные в формате json. Все эндпоинты принимают параметр ?lang (en|ru).

/api/gene/ — получение общего списка всех занесенных в базу генов с краткой информацией по каждому гену:

/api/gene/<id> — получение полной информации об одном гене по его id.

/api/gene/by-functional-cluster/<id>,<id>,... — получение списка генов, принадлежащих одновременно ко всем перечисленным функциональным классам. Структура ответа такая же, как в /api/gene/.

/api/gene/by-expression-change/<expression-change> — получение списка генов, имеющих заданное значение изменения экспрессии с возрастом. Структура ответа такая же, как в /api/gene/.

Project API returns data in json format. All endpoints take parameter ?lang (en|ru).

/api/gene/ — get general list of all genes in a base with short info on every gene:

/api/gene/<id> — get full info about one gene by its id.

/api/gene/by-functional-cluster/<id>,<id>,... — get a list of genes belonging simultaneously to all functional classes listed by their id's. The structure of response is the same as in /api/gene/.

/api/gene/by-expression-change/<expression-change> — get a list of genes with the same value of age-related changes in expression. The structure of response is the same as in /api/gene/.
